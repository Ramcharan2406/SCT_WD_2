## Stopwatch Web Application

An interactive and user-friendly stopwatch web application built using HTML, CSS, and JavaScript. This stopwatch allows users to accurately measure time intervals with start, pause, reset, and lap tracking functionality.


---

## Features

* Start,stop,pause and reset functionality

* Record lap times

* Responsive and has real-time updates

* Accurate time tracking using requestAnimationFrame



---

## Technologies Used

* HTML

* CSS

* JavaScript

##How it works

* JavaScript event listeners handle Start, Pause, Reset, and Lap button actions

* performance. now() is used to calculate precise elapsed time and requestAnimationFrame() ensures smooth and efficient timer updates

* The Lap feature dynamically creates and appends list items to record timestamps
